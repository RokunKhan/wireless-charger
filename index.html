Wireless Charger Using 555 Timer and MOSFET

By Rokun Khan

This project shows a basic wireless charger made with a 555 timer and a MOSFET. The 555 timer sends a fast on-off signal to the coil, creating a magnetic field. When another coil is placed near it, the power transfers wirelessly and lights up an LED. This shows how wireless energy transfer works using simple parts.

Process

  • Designed the circuit using a 555 timer in astable mode to create a square wave output.
  • Connected an RFP30N06LE MOSFET to amplify the pulse from the timer to the primary coil.
  • Built and placed the primary coil on the breadboard to emit a magnetic field.
  • Assembled the secondary coil with an LED to receive energy through induction.
  • Powered the circuit using a 9V battery and tested wireless transfer by lighting up the LED.
